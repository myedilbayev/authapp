# Приложение с AUTH авторизацией

## Введение

Это приложение позволяет пользователям авторизоваться с помощью Google и GitHub. 

**Технический стек:**
- Next.js
- NextAuth

## Установка

1. Установите зависимости:

   ```bash
   npm install



3. Запустите приложение локально используя команду в терминале:
    npm run dev
